🎯Complete Mental Health Platform

A comprehensive, production-ready application designed to provide enterprise-grade mental health support with advanced therapeutic features, security, and accessibility.

✨Core Features Implemented:

  • 🤖 AI First-Aid Chat - Deterministic trigger-based system with real-time crisis detection and safety protocols
  • 📊 Clinical Screenings - PHQ-9 and GAD-7 assessments with automated clinical scoring and interpretation
  • 👥 Confidential Booking - Anonymous counsellor session scheduling with privacy controls

  • 👑 Admin Panel - Comprehensive analytics dashboard accessible only at /admin

⚡Quick Start Guide

Get the full system running locally in minutes:

git clone https://github.com/opsaan/digital-psychological-intervention-system.git
cd digital-psychological-intervention-system
npm install
cp .env.example .env
npm run prisma:generate
npm run prisma:migrate
npm run prisma:seed
npm start

🎯Access Points:

  • Main Application: http://localhost:8080
  • Admin Dashboard: http://localhost:8080/admin (secure access only)

🔑Default Login Credentials:

  • System Administrator: admin@example.com / Admin123!@#
